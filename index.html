The Media Suggestion Portal!

This app allows you to search for a band name, book, movie or show title, or author name to find related media that matches your tastes

Find New Media Here!

Simply input a search term containing some band or author you like, and we'll handle the rest. If you want to input more than one entry, separate each entry with a comma. Sometimes it is useful to specify the type of a certain resource in the query. You can do this by prefacing your entry with one of the following: "band:", "movie:", "show:", "book:", "author:" or "game:". For example: "band:underworld, movie:harry potter"
